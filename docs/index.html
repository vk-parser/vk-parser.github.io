VkParser - это приложение, расширяющее возможности стандартного поиска людей в социальной сети ВКонтакте.

Среди дополнений к стандартным возможностям, можно выделить:
  • возможность найти людей, "скрытых" стандартным поиском ВК (см. Другие функции -> Многозапросный поиск)
  • возможность фильтрации по дате последнего посещения, по возможности написать сообщение и др.
  • возможность скрытия выбранных людей из результатов поиска (создание списков)
В общем, это незаменимый инструмент для маньяков и для тех, кто просто хочет познакомиться.
Условия использования, политика конфиденциальности, обратная связь, рекомендации и всё такое (там всё написанно для людей, можно читать).
