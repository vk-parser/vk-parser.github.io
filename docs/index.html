VkParser - это приложение (сайт), облегчающее поиск друзей и знакомства в вк (ВКонтакте).

Можно искать людей, которые были в сети недавно (заходил недавно). Можно скрыть закрытые профили и профили тех, кого нельзя добавить в друзья или кому нельзя писать сообщения в контакте. Можно найти скрытых вк пользователей (Многозапросный поиск) и создавать списки вк пользователей. Доступна сортировка по имени, по фамилии, по подписчикам, по возрасту и др. Парень вы или девушка, найти человека в вк с этим приложением будет проще. Всё бесплатно, без регистрации и без рекламы (мой подарок).
VkParser - незаменимый инструмент для маньяков и для тех, кто просто хочет познакомиться.
